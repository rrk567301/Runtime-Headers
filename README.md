# Runtime-Headers
A collection of dumped runtime headers.

## Contributing
If you would like to contribute, please take a look at my contribution guidelines first.

Additionally:
- Patch version dumps should not be submitted.
- Beta version dumps should not be submitted.
- [classdumpctl](https://github.com/leptos-null/classdumpctl) should be used for consistency.
